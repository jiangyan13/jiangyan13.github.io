强化学习 / 机器学习 / ROS仿真

我是一名在校硕士研究生，主要研究方向是机械臂的模仿学习与强化学习，并且熟悉ROS机械臂仿真和V-REP机械臂仿真，2021年完成基于ROS控制机械臂的项目。

“欲穷千里目，更上一层楼”，热爱研究各种新的语言与框架，目前涉及 Python / C++ / JS / Matlab 等编程语言。

目前正在进行基于UR5e机械臂研究模仿学习方法和应用技术，主要寻找最优奖励函数进一步将深度学习、模仿学习与机械臂控制技术融合。

同时强化学习也是研究重点之一，研究机器人如何通过与环境交互来改善其策略，目标是创建数据高效的机器学习算法，使得这些算法可以适用于复杂环境机器人领域，让机器人变得更智能化。
