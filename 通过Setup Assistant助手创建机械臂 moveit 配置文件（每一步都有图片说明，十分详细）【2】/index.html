前言

一、详细步骤介绍

1.启动 setup assistant 助手

2.导入机械臂模型的 urdf 文件

点击 Create New MoveltConfiguration Package ，导入 urdf 文件：

点击load files 载入文件

3.Self Collisions 配置

点击 Generate collision Matrix，生成碰撞矩阵：

4.虚拟关节配置

按照下图修改名称：

修改完成后点击save，生成虚拟关节：

5.Planning Groups 配置

点击Add Group，添加运动规划组：

按照下图进行配置Kinematic Solver：

点击Add Joints添加关节，把机械臂的六个关节选中，点击save：

6.Define Robot Poses 配置

点击Add Poses，定义机械臂姿态：

点击save保存：

7.Define Robot Poses 配置

此次配置没有夹爪，故不进行配置End Effectors和Passive Joints，直接跳过即可：

8.ROS Control 配置

点击 Add Controller 添加控制器，
注意 Controller Type 选择 ==position_controllers/JointTrajectoryController==：

点击 Add Planning Group Joints ，添加 arm：

点击save保存，ros controller配置完成

9.Simulation 配置

点击Generate URDF生成 gazebo 仿真时所需的 urdf 文件
把此URDF文件保存到新建文档中，命名为 ==ur5e.urdf==
==这一步特别重要，不然后面 gazebo 不能仿真==

10.Simulation

3D Perception 跳过即可，Author Information 随便填写

11.生成配置文件

先选择配置文件保存的位置，然后点击Configuration Files生成配置文件

==Note，需要在工程src路径下新建一个文件夹（例如命名为ur5_moveit_config）存放生成的配置文件，不是新建功能包，否则会冲突报错。==
